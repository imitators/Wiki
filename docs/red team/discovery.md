## 系统网络配置发现

IP、网关、DNS、是否能连通外网、网络连接和端口、本机host文件、机器代理、是否在域内、域信息

TCP 协议

```
vps: nc -lvvp 80
target: curl [vps_ip] 80
```

ICMP 协议

```
vps: tcpdump icmp
target: ping [vps_ip]
```

DNS 协议

```
vps:nc -u -lvp 53
target:
nslookup www.baidu.com [vps_ip]
dig @[vps_ip] www.baidu.como
```

端口判断

外网做vps监听，内网机器测试常见端口，例如 80,8080,443,53,110,123...
