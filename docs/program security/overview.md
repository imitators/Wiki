# 导航

基于未来是融合发展的判断，所以添加了**融合学科**这一新概念。

因此 Program Security 大类融合了以下学科：

1. 软件工程
2. 计算机网络
3. 计算机组成原理
4. 操作系统
5. 网络安全研究
